# HSE Admission Mobile

Мобильное приложение приемной комиссии НИУ ВШЭ (Нижний Новгород) на Kotlin + Jetpack Compose.

## Возможности
- Локальная регистрация и вход (Room)
- Сессия пользователя (DataStore)
- Экраны: Splash, Login, Register, Home, Profile, Application, Admin
- Подача заявления с выбором факультета, образовательной программы, формы обучения и PDF-файла
- Полностью офлайн
- Material 3, MVVM, Hilt, Navigation Compose

## Стек
Kotlin, Jetpack Compose, Room, DataStore, Hilt, Coroutines, StateFlow.

## Запуск
1. Открыть проект в Android Studio Iguana+.
2. Sync Gradle.
3. Запустить `app` на Android 8+.
